## xml文件特殊字符处理

1. CDATA

```xml
<![CDATA["{{&&&&&&<<<<<<<}}"]]>
```

2. 转义字符

| 转义字符 | 符号 | 名称 |
| --- | --- | --- |
| &amp;	  | &	| 和号 |
| &lt;	  | <	| 小于 |
| &gt;	  | >	| 大于 |
| &apos;  | '	| 省略号 |
| &quot;  | "	| 引号 |
