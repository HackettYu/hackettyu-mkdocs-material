- [文件描述符](https://zh.wikipedia.org/wiki/%E6%96%87%E4%BB%B6%E6%8F%8F%E8%BF%B0%E7%AC%A6)
文件描述符（File descriptor）是计算机科学中的一个术语，是一个用于表述指向文件的引用的抽象化概念。
- linux使用的进程间通信方式
    - 管道（pipe）,流管道(s_pipe)和有名管（FIFO）
    - 信号（signal）
    - 消息队列
    - 共享内存
    - 信号量
    - 套接字（socket)
